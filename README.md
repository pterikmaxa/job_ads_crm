# Job Ads CRM

Личная CRM-система для управления поиском работы в Германии и Украине.

## Текущая среда разработки

- Python: см. `.venv`
- Django: 6.0.6
- Database: SQLite на этапе MVP
- OS: Ubuntu
- IDE: VS Code + Codex

## Назначение проекта

Проект используется для хранения CV-профилей, сайтов вакансий, поисковых сигнатур и объявлений о работе.

На первом этапе объявления добавляются вручную. Позже будет добавлен crawler/spider для автоматического сбора вакансий по поисковым сигнатурам.
